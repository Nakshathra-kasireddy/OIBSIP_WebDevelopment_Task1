# 🌐 Landing Page Project

## 📌 Task Overview
This project is a simple and responsive **Landing Page** built using HTML and CSS.  
It is designed as a beginner-friendly web development project to understand layout design, section structuring, and styling techniques.

---

## 🎯 Objective
The main objective of this project is to:
- Understand the structure of a web page
- Learn how to divide content into sections
- Practice HTML layout elements
- Improve CSS styling and alignment skills
- Build a visually appealing landing page

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- VS Code (Editor)

---

## 📚 Features
- Responsive layout structure
- Header section with navigation
- Multiple content sections (columns and boxes)
- Footer section
- Proper alignment and spacing
- Attractive color combination and design
- Clean and simple UI

---

## 🚀 What I Learned
- How to structure a webpage using HTML
- How to divide sections using divs and semantic tags
- How to use CSS for styling, spacing, and positioning
- Importance of padding, margins, and alignment
- How to create a visually balanced layout
